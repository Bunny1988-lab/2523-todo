Front-End Engineering (FEE - 2) Practical Assignment 

 

Project Title: Todo List Application using HTML, CSS & JavaScript 

Objective: Build a task management application where users can add, view, edit, delete, and complete tasks. The application should store data permanently using browser localStorage. 

User Story 

As a user, I want to manage my daily tasks using a Todo List application so that I can add new tasks, update existing tasks, mark tasks as completed, and remove tasks that are no longer required. 

Project Requirements 

Create a responsive Todo List user interface using HTML and CSS. 

Use JavaScript to add dynamic functionality. 

Store tasks using JavaScript arrays and objects. 

Implement CRUD operations. 

Store tasks in localStorage so data remains after refreshing the page. 

Upload the complete project code to GitHub. 

Step-by-Step Development Instructions 

Step 1: Create Project Structure 

Create a project folder with the following files: 
 
index.html 
style.css 
script.js 

Step 2: Create HTML Structure 

Create the basic layout: 
 
- Application heading 
- Input field for entering tasks 
- Add button 
- Unordered list (ul) to display tasks 

Step 3: Add CSS Styling 

Style the application: 
 
- Center the Todo container 

- Design input and buttons 
- Create task item styles 

Step 4: Connect JavaScript With HTML 

Select HTML elements using DOM methods such as getElementById(). Add event listeners for button clicks. 

Step 5: Create Add Task Functionality 

When the user enters a task and clicks Add: 
 
- Read input value 
- Validate empty input 
- Store task data in an array 
- Display the task on screen 

Step 6: Use Modular JavaScript Structure 

Separate code into functions: 
 
- addTodo() - Add new tasks 
- renderTodo() - Display tasks 
- createTodoItem() - Create task HTML elements 

Step 7: Implement Delete Feature 

Add a Delete button for every task. 
 
When clicked: 
 
- Identify task index 
- Remove task from array 
- Update the UI 

Step 8: Implement Edit Feature 

Add an Edit button. 
 
When clicked: 
 
- Get existing task value 
- Allow user to modify it 
- Update task data 
- Re-render the list 

Step 9: Add Complete Task Feature 

Add a checkbox for every task. 
 
When checked: 
 
- Update completed status 
- Apply completed styling using CSS 

Step 10: Add Local Storage 

Store Todo data permanently. 
 
Use: 
 
- localStorage.setItem() 
- localStorage.getItem() 
- JSON.stringify() 
- JSON.parse() 
 
Load saved tasks when the application starts. 

Expected Final Features 

Add new tasks 

Display all tasks 

Edit tasks 

Delete tasks 

Mark tasks as completed 

Save tasks after browser refresh 

Responsive user interface 

Upload Project Code on GitHub 

1. Create a new repository on GitHub with a meaningful name (example: todo-list-javascript). 

2. Open the project folder in VS Code. 

3. Initialize Git using: git init 

4. Add all files using: git add . 

5. Create your first commit using: git commit -m "Create Todo List Application" 

6. Connect your local project with GitHub repository using the remote URL. 

7. Push your code using: git push -u origin main 

8. Verify that all project files are visible on GitHub.
